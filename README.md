NSQ in Rust
===========

[NSQ](https://nsq.io/) client implementation in pure Rust.

## Features

* Pure Rust
* Async implementation based on [Futures 0.3](https://docs.rs/futures/0.3) and [Tokio 0.2](https://docs.rs/tokio/0.2)
* High level
    * [x] Producer(PUB)
    * [ ] Comsumer(SUB)
    * [ ] Discovery
    * [ ] Backoff
    * [x] TLS
    * [x] Snappy
    * [ ] Sampling
    * [x] AUTH
